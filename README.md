# Number Tools (Java)

A console-based Java application that provides several useful number operations such as checking even/odd numbers, finding the maximum of two numbers, reversing numbers, calculating the sum of digits, and checking palindrome numbers.

This project was built to practice **Java methods, loops, conditionals, and modular program structure**.

---

## Features

* Check if a number is **Even or Odd**
* Find the **maximum of two numbers**
* **Reverse a number**
* Calculate the **sum of digits**
* Check whether a number is a **palindrome**
* Menu-driven CLI program
* Modular design using **separate methods**

---

## Concepts Practiced

* Java Methods
* Parameters and Return Values
* Conditional Statements (`if`, `else`)
* Loops (`while`)
* Arithmetic Operators
* Modular Programming
* `Scanner` class for user input

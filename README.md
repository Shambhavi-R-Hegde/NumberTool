# Number Tools (Java)

A console-based Java application that provides several useful number operations using modular methods.

This project was created to practice Java **methods, loops, conditionals, and structured programming**.

---

## Features

The program includes the following utilities:

1. Check Even or Odd
2. Find Maximum of Two Numbers
3. Reverse a Number
4. Sum of Digits
5. Palindrome Check
6. Prime Number Check
7. Count Digits in a Number

The program runs in a menu-driven CLI interface.

---

## Concepts Practiced

- Java Methods
- Method Parameters
- Return Values
- Loops (`while`, `for`)
- Conditional Statements (`if`, `else`)
- Arithmetic Operators
- Scanner for user input
- Modular Programming
